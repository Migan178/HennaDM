# MF

-   This project is template and light-weight framework for [discordgo](https://github.com/bwmarrin/discordgo).
-   This project is based by a discord bot [muffinBot](https://github.com/Muffin-laboratory/goMuffin).

# Features

-   builder-pattern
-   command and message component, modal components loader (commands/discommand.go)
-   paginated container (ComponentsV2, It looks like embed.)

## Examples

-   Command example: commands/ping.go (commands/discord.go is command loader)

-   Message Component example: components/pagination_container.go

-   Modal Component example: modals/pagination_container.go

# Configure

-   You should set `.env` file to set bot configs. You can copy contents `.env.example` file's in `.env`.
